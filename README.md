# E-Transfer Application

Spring Boot based P2P money transfer application for interview preparation.

## Features
- User profile lookup
- Balance inquiry
- Money transfer (max $3000)
- Caching
- Transaction consistency

## Tech Stack
- Java
- Spring Boot
- JPA
- H2/Postgres
